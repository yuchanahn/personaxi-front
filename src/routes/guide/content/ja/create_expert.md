# エキスパートモード（Expert Mode）

ディテール、その先の完璧さへ。🛠️
プロンプトエンジニアリングを通じてキャラクターの魂まで設計したい方のための高度なツールです。

## 🏗️ 主要機能

### 1. システムプロンプト（System Prompt）
AIの行動原則を直接制御します。
*   `{{user}}`、`{{char}}`変数を使用して動的に名前が変わるよう設定できます。
*   マークダウン（Markdown）記法を使って読みやすい指示を出せます。

### 2. 対話例（Few-shot Examples）
「百聞は一見にしかず。」AIに話し方を学習させる最も確実な方法です。
*   `<START>`タグで区切って複数の対話例を入力してください。
*   キャラクター特有の話し方、感嘆詞、ト書きを含めると完璧に模倣します。

### 3. ナレッジベース（KB / RAG）
キャラクターが知るべき膨大な設定や世界観文書をアップロードしてください。
*   キャラクターが会話中に該当情報を「検索」して回答に反映します。
*   *例：ファンタジー小説の魔法設定集、歴史的事件の年表など*

### 4. 高度な設定（Parameters）
*   **Temperature**：創造性の調整（高いほど型破りで創造的、低いほど論理的で反復的）
*   **Top P / Top K**：回答の多様性コントロール
