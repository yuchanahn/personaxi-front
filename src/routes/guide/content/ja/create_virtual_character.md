# バーチャルモデルアップロードガイド

単なる画像を超えて、動くキャラクターをペルソナに連携する方法です。💃
**Live2D**と**VRM（3D）**ファイルに対応しています。

## 📤 アップロード方法

キャラクター作成/編集画面の**「アバター/モデル」**タブからファイルをアップロードできます。

### 1. Live2Dモデル
*   **対応ファイル**：`.model3.json`ファイルを含むフォルダ構造、または`.zip`パッケージ
*   **必須ファイル**：`model3.json`、`.moc3`、テクスチャファイルなど一式が必要です。
*   **モーション**：デフォルトの待機モーション（Idle）と表情変化が含まれていれば自動認識されます。

### 2. VRM（3D）モデル
*   **対応ファイル**：`.vrm`（VRM 0.x および 1.0 標準対応）
*   **特徴**：標準ヒューマノイドボーン（Bone）構造に従うモデルであればほぼ互換性があります。

## ⚠️ 注意事項

*   **容量制限**：快適なロードのため**30MB以下**のファイル使用を推奨します。
*   **著作権**：他者のモデルを無断でアップロードすると著作権侵害で制裁される場合があります。（Boothなどで購入したモデルのライセンスを必ず確認してください！）

> [!TIP]
> **サムネイル設定**
> モデルをアップロードしても、リストに表示される**2Dサムネイル画像**は別途設定することをお勧めします。（モデルロード前にユーザーに表示されます。）
