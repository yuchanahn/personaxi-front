# バーチャルモデルアップロードガイド

2.5D（Live2D）/ 3D（VRM）モデルをキャラクターに接続する方法です。

## アップロード場所

- `edit3` のメディア/モデル領域でアップロード
- Live2D は保存後に **Live2D 高級編集** でモーション/表情/音声/ランタイムを調整

## 1) Live2D

- **推奨入力**：`.zip` パッケージまたは `.model3.json` 一式
- **必須ファイル**：`model3.json`, `.moc3`, テクスチャ, モーション/表情リソース
- **高級編集可能項目**:
  - モーション/表情エイリアス
  - AIアクションマッピング
  - 音声マッピング
  - ランタイム感度/オフセット

## 2) VRM (3D)

- **対応ファイル**：`.vrm`（0.x / 1.0）
- **推奨条件**：標準ヒューマノイド構造、過度なテクスチャ/ポリゴンを避ける

## 運用のコツ

- モデルとは別にサムネイル画像を設定するのがおすすめです。
- モデル差し替え後はモーション/表情/音声をテストチャットで再確認してください。
- ライセンス（再配布・商用利用制限）を先に確認してください。
