# 通知設定（Notification）

{{PUBLIC_BRAND_NAME}}の重要なお知らせを見逃さないように通知を設定しましょう。🔔

## 📢 通知の種類

| アイコン | 種類 | 説明 |
| :---: | :--- | :--- |
| ❤️ | **アクティビティ通知** | マイキャラクターに「いいね」が押されたり、コメントが付いた時にお知らせします。 |
| 💬 | **チャット通知** | （予定）キャラクターが先に話しかけたり、チャットイベントを送る時に受け取ります。 |
| 🎉 | **マーケティング通知** | 新機能、イベント、無料ニューロン付与などの特典情報を受け取ります。 |

## ⚙️ 設定方法

1.  `設定` > `通知`メニューに移動します。
2.  ご希望の通知項目を<span style="color:#22c55e">ON</span> / <span style="color:#ef4444">OFF</span>に切り替えられます。

> [!NOTE]
> **Webプッシュ通知**
> 通知はブラウザのプッシュ機能を使用します。ブラウザで「通知の許可」を**許可**に設定する必要があります。
