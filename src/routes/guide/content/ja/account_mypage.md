# プロフィールとアカウント管理

**マイページ（My Page）**でご自身の活動履歴を確認し、他のユーザーに表示されるプロフィールをカスタマイズできます。✨

## 📝 プロフィール編集

`設定` > `プロフィール編集`メニューから以下の情報を変更できます。

| 項目 | 説明 |
| :--- | :--- |
| **ニックネーム** | コミュニティでの活動時に表示される名前です。 |
| **クリエイタータグ** | あなた固有のIDです。（`@username`形式）このタグで他のユーザーがあなたを検索できます。 |
| **プロフィール画像** | 個性を表す画像をアップロードしてください。（GIF対応予定） |
| **自己紹介** | 自分を表現する一行紹介を入力してみてください。 |

## 📊 活動履歴の確認

*   **マイペルソナ**：自作のキャラクター一覧を管理します。
*   **いいねしたペルソナ**：お気に入り登録したキャラクターをまとめて閲覧します。
*   **フォロー/フォロワー**：興味のあるクリエイターとの関係を確認します。

> [!TIP]
> **クリエイタータグの変更**
> クリエイタータグは固有のアドレスとして使用されるため、慎重に決定してください！変更すると既存のプロフィールリンクが機能しなくなる場合があります。
