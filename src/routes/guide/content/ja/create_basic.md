# 基本作成ガイド（Basic Mode）

複雑なプロンプト設計なしで、最短でキャラクターを作るモードです。  
保存後は `edit` で詳細設定（プロンプト・変数・高級編集）を続けられます。

## 作成フロー

### 1. 基本情報
- **名前**：表示名
- **プロフィール画像**：一覧サムネイル
- **カテゴリタグ**：ジャンルと雰囲気

### 2. キャラクターのコア設定
- **紹介文**：カードに表示される短文
- **口調/性格**：会話の基本トーン
- **最初のシーン (`first_scene`)**：チャット開始時に表示される本文

### 3. 最初のシーン作成のコツ
- 地の文と最初のセリフを同じブロックに入れると自然です。
- アセット画像を入れる場合は `<img 0>`, `<img 1>` を使用します。
- 例:
  - `深夜、窓辺に立つ彼女がゆっくり振り向く。`
  - `<say speaker="{{char}}">来たんだね。今夜はどこから始めようか。</say>`

### 4. 公開範囲
- **公開（Public）**：検索/共有可能
- **非公開（Private）**：テスト向け

## 保存後の推奨作業

- `edit` でプロンプトを補強
- 必要なら Live2D/VRM を高級編集で接続
- 2Dキャラは変数/インタラクティブ規則を確認

> [!NOTE]
> Live2D/VRM の接続は作成直後でも後からでも可能です。
