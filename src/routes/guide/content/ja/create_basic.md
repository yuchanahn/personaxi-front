# 基本制作ガイド（Basic Mode）

複雑な設定なしで、誰でも簡単に素早くペルソナを作れるモードです。
**たった4ステップ**であなただけのAI友達が誕生します！🥚 ➡️ 🐣

## 🛠️ 制作ステップ

### 1. 基本情報（Identity）
*   **名前**：キャラクターに名前をつけましょう。
*   **プロフィール画像**：キャラクターを最もよく表す画像をアップロードしてください。
    *   （ヒント）顔がよく見える画像の方がクリック率が高いです。
*   **カテゴリー**：ファンタジー、ロマンス、学園などジャンルを選択します。

### 2. ペルソナ設定（Persona）
キャラクターの性格を決める最も重要なステップです。
*   **性格タグ**：`ツンデレ` `優しい` `クール` などキーワードを選ぶとAIが自動で性格を形成します。
*   **紹介文**：キャラクターリストに表示される短いフレーズです。（例：「あなたの幼馴染、でも秘密がある。」）

### 3. 初めての挨拶（First Message）
ユーザーがチャットルームに初めて入った時、キャラクターが交わす最初の一言です。
*   状況を描写したり（ト書き）、話しかける形で書いてみてください。
*   *例：「（本を読んでいたが顔を上げて）あ、来たんだ？待ってたよ。」*

### 4. 公開設定
*   **公開（Public）**：誰でも検索して会話できます。（インセンティブ対象）
*   **非公開（Private）**：自分だけが会話できます。テスト用におすすめです。
