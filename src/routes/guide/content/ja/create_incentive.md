# クリエイター貢献報酬（Contribution Reward）

クリエイティブなキャラクター制作でサービスの活性化に貢献されたクリエイターの皆様に、感謝の意を込めて報酬を提供します。🎁
{{PUBLIC_BRAND_NAME}}は創作者の努力が尊重される健全なエコシステムをともに作っていきます。

## 📈 貢献度の算定方式

制作されたキャラクターがユーザーとインタラクションし、サービスの価値を高めるたびに貢献ポイントが蓄積されます。

* **算定基準：** キャラクターインタラクション指数およびサービス活性への貢献度に基づく
* **報酬比率：** システム貢献度に応じて算出された一定比率の**クリエイター貢献度（無償）**を付与

## 🎁 報酬活用および精算ポリシー

蓄積された「クリエイター貢献度」はサービス内部の創作貢献指標として活用されます。

### 1. 内部貢献指標
* 貢献度はキャラクターのサービス貢献度を示す内部指標として管理されます。
* 表示順位、推薦、パートナー選定などの運営指標として活用される場合があります。

### 2. パートナー精算プログラム（準備中）
* プラットフォーム貢献度が高い優秀なクリエイターを対象とした**「パートナー精算プログラム」**を準備中です。
* このプログラムは単純なポイント換金ではなく、**コンテンツ供給契約に基づく事業/その他所得精算**方式で運営される予定であり、関連法令に基づく透明な精算手続きを遵守します。

> [!TIP]
> **貢献度を高めるための創作ガイド**
> * 完成度の高いペルソナと魅力的なビジュアル素材でユーザーの没入感を高めましょう。
> * **最初のインタラクション（First 3-Turns）**の設計が重要です！興味深い会話の始まりを誘導しましょう。
> * 定期的なキャラクターアップデートを通じてユーザーと継続的にコミュニケーションしましょう。
