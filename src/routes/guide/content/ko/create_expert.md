# 전문가 모드 (Expert Mode)

디테일, 그 이상의 완벽함. 🛠️
프롬프트 엔지니어링을 통해 캐릭터의 영혼까지 설계하고 싶은 분들을 위한 고급 도구입니다.

## 🏗️ 주요 기능

### 1. 시스템 프롬프트 (System Prompt)
AI의 행동 원칙을 직접 제어합니다.
*   `{{user}}`, `{{char}}` 변수를 사용하여 동적으로 이름이 바뀌도록 설정할 수 있습니다.
*   마크다운(Markdown) 문법을 사용하여 가독성 있게 지침을 내릴 수 있습니다.

### 2. 대화 예시 (Few-shot Examples)
"백문이 불여일견." AI에게 말투를 학습시키는 가장 확실한 방법입니다.
*   `<START>` 태그로 구분하여 여러 개의 대화 예시를 입력하세요.
*   캐릭터 특유의 말투, 감탄사, 행동 지문을 포함하면 완벽하게 모방합니다.

### 3. 지식 베이스 (KB / RAG)
캐릭터가 알아야 할 방대한 설정이나 세계관 문서를 업로드하세요.
*   캐릭터가 대화 중에 해당 정보를 "검색"하여 답변에 반영합니다.
*   *예: 판타지 소설의 마법 설정집, 역사적 사건 연표 등*

### 4. 고급 설정 (Parameters)
*   **Temperature**: 창의성 조절 (높을수록 엉뚱하고 창의적, 낮을수록 논리적이고 반복적)
*   **Top P / Top K**: 답변 다양성 제어
