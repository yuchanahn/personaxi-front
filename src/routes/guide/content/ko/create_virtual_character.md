# 버추얼 모델 업로드 가이드

단순 이미지를 넘어, 움직이는 캐릭터를 페르소나에 연결하는 방법입니다. 💃
**Live2D**와 **VRM(3D)** 파일을 지원합니다.

## 📤 업로드 방법

캐릭터 생성/수정 화면의 **'아바타/모델'** 탭에서 파일을 업로드할 수 있습니다.

### 1. Live2D 모델
*   **지원 파일**: `.model3.json` 파일이 포함된 폴더 구조, 또는 `.zip` 패키지
*   **필수 파일**: `model3.json`, `.moc3`, 텍스처 파일 등 전체 세트가 필요합니다.
*   **모션**: 기본 대기 모션(Idle)과 표정 변화가 포함되어 있다면 자동으로 인식합니다.

### 2. VRM (3D) 모델
*   **지원 파일**: `.vrm` (VRM 0.x 및 1.0 표준 지원)
*   **특징**: 표준 휴머노이드 본(Bone) 구조를 따르는 모델이라면 대부분 호환됩니다.

## ⚠️ 유의사항

*   **용량 제한**: 쾌적한 로딩을 위해 **30MB 이하**의 파일 사용을 권장합니다.
*   **저작권**: 타인의 모델을 허락 없이 업로드하면 저작권 위반으로 제재될 수 있습니다. (Booth 등에서 구매한 모델의 라이선스를 꼭 확인하세요!)

> [!TIP]
> **썸네일 설정**
> 모델을 업로드하더라도, 목록에 표시될 **2D 썸네일 이미지**는 별도로 설정하는 것이 좋습니다. (모델 로딩 전 유저들에게 보여집니다.)
