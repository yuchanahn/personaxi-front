# 버추얼 모델 업로드 가이드


2.5D(Live2D) / 3D(VRM) 캐릭터 연결 방법입니다.

## 업로드 위치

- 캐릭터 편집(`edit`)의 미디어/모델 영역에서 업로드
- Live2D는 저장 후 **Live2D 고급 편집** 페이지에서 모션/표정/오디오/런타임 값을 추가 조정

## 1) Live2D

- **권장 입력**: `.zip` 패키지 또는 `.model3.json` 포함 세트
- **필수 구성**: `model3.json`, `.moc3`, 텍스처, 모션/표정 파일
- **고급 편집 가능 항목**:
  - 모션/표정 별칭(alias)
  - AI 액션 매핑
  - 오디오 연결
  - 런타임 민감도/오프셋

## 2) VRM (3D)

- **지원 파일**: `.vrm` (0.x / 1.0)
- **권장 조건**: 표준 휴머노이드 구조, 과도한 폴리곤/텍스처 지양

## 운영 팁

- 썸네일은 모델과 별도로 설정하는 것이 좋습니다.
- 모델 교체 후에는 테스트 채팅에서 모션/표정/음성 동작을 반드시 재확인하세요.
- 라이선스 문구(2차 배포 금지, 상업 이용 제한 등)를 먼저 확인하세요.
