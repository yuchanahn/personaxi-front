# 프로필 및 계정 관리

**내 페이지(My Page)**에서 나의 활동 내역을 확인하고, 다른 유저들에게 보여질 모습을 꾸밀 수 있습니다. ✨

## 📝 프로필 편집

`설정` > `프로필 편집` 메뉴에서 다음 정보를 수정할 수 있습니다.

| 항목 | 설명 |
| :--- | :--- |
| **닉네임** | 커뮤니티 활동 시 표시되는 이름입니다. |
| **크리에이터 태그** | 나만의 고유한 ID입니다. (`@username` 형식) 이 태그로 다른 유저가 나를 검색할 수 있습니다. |
| **프로필 이미지** | 개성을 나타내는 이미지를 업로드하세요. (GIF 지원 예정) |
| **소개글** | 나를 표현하는 한 줄 소개를 입력해보세요. |

## 📊 활동 내역 확인

*   **내가 만든 페르소나**: 직접 제작한 캐릭터 목록을 관리합니다.
*   **좋아요한 페르소나**: 마음에 들어서 '하트'를 누른 캐릭터들을 모아봅니다.
*   **팔로잉/팔로워**: 관심 있는 크리에이터와의 관계를 확인합니다.

> [!TIP]
> **크리에이터 태그 변경**
> 크리에이터 태그는 고유한 주소로 사용되므로, 신중하게 결정해주세요! 변경 시 기존 프로필 링크가 작동하지 않을 수 있습니다.
