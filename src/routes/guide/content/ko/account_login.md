# 로그인 및 문제 해결

## 🔑 로그인 방법

가입 시 사용했던 방법과 동일한 수단을 선택해주세요.

*   **구글 가입자**: `Google로 계속하기` 버튼 클릭
*   **카카오 가입자**: `Kakao로 시작하기` 버튼 클릭
*   **이메일 가입자**: 이메일과 비밀번호 입력

---

## 💡 자주 묻는 질문 (FAQ)

### Q. 로그인이 안 돼요!
*   가입했던 소셜 계정이 맞는지 확인해주세요. (예: 구글로 가입하고 카카오로 로그인 시도 시, 새로운 계정이 생성될 수 있습니다.)
*   브라우저의 쿠키/캐시를 삭제하고 다시 시도해보세요.

### Q. 비밀번호를 잊어버렸어요.
*   **소셜 로그인** 유저는 해당 플랫폼(Google, Kakao)에서 비밀번호를 관리합니다.
*   **이메일 가입** 유저는 로그인 화면 하단의 `비밀번호 찾기`를 이용해주세요.
