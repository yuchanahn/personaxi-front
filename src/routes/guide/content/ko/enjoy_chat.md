# 채팅 모드 안내 (Chat Modes)


PersonaXi는 일반 대화와 인터랙티브 시뮬레이션을 모두 지원합니다.

## 1) 캐릭터 채팅 (1:1)

- 한 캐릭터와 관계 중심 대화를 하는 기본 모드
- 대화 누적에 따라 이벤트/히든 이미지가 열릴 수 있음
- TTS, 이미지 출력, 대사 태그 기반 렌더링 지원

## 2) 스토리/시뮬레이션 채팅

- 상태 변화, 선택지, 입력창을 포함한 게임형 대화 가능
- 제작자가 규칙을 설정하면 상태창/변수 기반 진행이 가능

## 2D 인터랙티브 출력이란?

2D 채팅에서는 캐릭터가 HTML/CSS + 버튼/입력 UI를 출력할 수 있습니다.

- 버튼 예: `.game-choice`
- 선택 횟수 카운터: `.game-choice-counter`
- 자유 입력: `.game-input`
- 턴 종료: `.game-input-end`

사용자는 버튼 클릭과 입력을 마친 뒤 제출하며, 시스템은 이 입력을 요약해 AI에 다시 전달합니다.

## 변수 시스템

- 2D 캐릭터는 응답에 `<vars>...</vars>` 블록을 포함해 상태값을 갱신할 수 있습니다.
- 블록 내부 형식은 `key=value` 줄 단위입니다.
- 이 값은 상태 패널 표시와 다음 턴 추론에 사용됩니다.

## 대사/이미지 출력 형식

- 대사: `<dialogue speaker="이름">...</dialogue>`
- 에셋 이미지: `<img 0>` 형식
- 마크다운/HTML 스타일 텍스트는 안전 필터링 후 렌더링

> [!TIP]
> 인터랙티브 기능이 있는 캐릭터를 만들고 싶다면 `전문가 모드(create_expert)`에서 출력 규칙을 먼저 설계한 뒤 테스트 채팅으로 검증하세요.
