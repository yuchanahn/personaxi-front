# 기본 제작 가이드 (Basic Mode)


복잡한 프롬프트 작성 없이 빠르게 캐릭터를 만드는 모드입니다.  
저장 후에는 `edit3`에서 세부 설정(프롬프트, 변수, 고급 편집)까지 이어서 조정할 수 있습니다.

## 제작 순서

### 1. 기본 정보
- **이름**: 검색/채팅 목록에 표시될 캐릭터 이름
- **프로필 이미지**: 목록 썸네일
- **카테고리 태그**: 캐릭터 장르와 분위기

### 2. 캐릭터 핵심 설정
- **소개글**: 카드에 노출되는 짧은 설명
- **성격/톤**: 말투 방향을 정하는 기본 성격
- **첫 장면(first_scene)**: 대화 시작 직후 출력되는 장면 텍스트

### 3. 첫 장면 작성 팁
- 장면 설명 + 첫 대사를 같이 넣으면 자연스럽습니다.
- 에셋 이미지를 연결하려면 본문에 `<img 0>`, `<img 1>` 같은 태그를 넣으세요.
- 예시:
  - `늦은 밤, 창가에 기대 선 그녀가 천천히 고개를 돌린다.`
  - `<dialogue speaker="{{char}}">왔네. 오늘은 어디부터 이야기할까?</dialogue>`

### 4. 공개 범위
- **공개(Public)**: 검색/추천 노출
- **비공개(Private)**: 테스트 및 개인용

## 저장 후 추천 작업

- `edit3`에서 프롬프트 탭 보강
- 필요 시 고급 편집으로 Live2D/VRM 연결
- 2D 캐릭터라면 변수/인터랙티브 규칙 점검

> [!NOTE]
> Live2D/VRM 연결은 생성 직후에도 가능하며, 이후 언제든 수정 페이지에서 다시 조정할 수 있습니다.
