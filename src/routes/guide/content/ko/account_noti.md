# 알림 설정 (Notification)

PersonaXi의 중요한 소식을 놓치지 않도록 알림을 설정해보세요. 🔔

## 📢 알림 종류

| 아이콘 | 종류 | 설명 |
| :---: | :--- | :--- |
| ❤️ | **활동 알림** | 내 캐릭터에 '좋아요'가 눌리거나, 댓글이 달렸을 때 알려줍니다. |
| 💬 | **대화 알림** | (예정) 캐릭터가 나에게 먼저 말을 걸거나, 대화 이벤트를 보낼 때 받습니다. |
| 🎉 | **마케팅 알림** | 새로운 기능, 이벤트, 무료 뉴런 지급 등 혜택 정보를 받습니다. |

## ⚙️ 설정 방법

1.  `설정` > `알림` 메뉴로 진입하세요.
2.  원하는 알림 항목을 <span style="color:#22c55e">ON</span> / <span style="color:#ef4444">OFF</span> 할 수 있습니다.

> [!NOTE]
> **웹 푸시 알림 (Web Push)**
> 알림은 브라우저의 푸시 기능을 사용합니다. 브라우저에서 '알림 권한'을 **허용**으로 설정해야 정상적으로 수신됩니다.
