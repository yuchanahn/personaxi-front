# Notification Settings

Don't miss out on important news from PXI. 🔔

## 📢 Notification Types

| Icon | Type | Description |
| :---: | :--- | :--- |
| ❤️ | **Activity** | Notifies when your character gets a 'Like' or a comment. |
| 💬 | **Chat** | (Coming Soon) Receive messages initiated by characters or event notifications. |
| 🎉 | **Marketing** | Get news on new features, events, and free Neuron benefits. |

## ⚙️ How to Configure

1.  Go to `Settings` > `Notifications`.
2.  Toggle <span style="color:#22c55e">ON</span> / <span style="color:#ef4444">OFF</span> for each item.

> [!NOTE]
> **Web Push**
> Notifications use browser push technology. You must **Allow** notifications in your browser settings to receive them.
