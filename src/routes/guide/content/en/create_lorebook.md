# Lorebook System Guide

> **Note**: This guide is currently being translated. Please refer to the Korean version or check back later.

## What is a Lorebook?

A Lorebook (or World Info) is a dynamic memory system that allows your AI character to access a vast amount of background information only when it's relevant to the conversation.

## Key Features

1.  **Keyword Activation**: Information is injected only when specific keywords appear in the chat.
2.  **Priority System**: Important information takes precedence when space is limited.
3.  **Unlimited Entries**: You can create as many entries as you want!

## Constraints

*   **Max Length per Entry**: 500 characters
*   **Token Budget**: Approx 2000 tokens per turn
